Projet virtualisation

Mini-labo VMware
avec Firewall

Réseau virtuel isolé + filtrage + tests de communication

Description

Création d'un réseau isolé contenant deux machines virtuelles reliées à un firewall. Objectif : tester des règles de filtrage et observer les communications entre machines.

Objectifs techniques

  • Gestion de VMs
  • Réseau interne isolé
  • Filtrage via firewall
  • Tests de connectivité
  • Ping / Ports
